# Бухгалтерия

## Назначение

Чат «Бухгалтерия» — отдельная Telegram-группа куда юзербот пишет ежедневные финансовые отчёты, а менеджер вручную вводит расходы/курсы/правки.

ID хранится в `storage.accounting_group_id`. Задаётся через `/admin → 🧠 AI → 📊 Бухгалтерия чат ID`.

## Что считается за день

Чистая маржа = (оборот − выплаты партнёрам − стоимость ЛК + ручные правки) / 1.17

Из чистой маржи: операционист 15%, откупщик 2%.

### Компоненты:

- **Оборот** (turnover) — сумма прошедшая по счетам клиентов в рублях. Авто-записывается при `update_deal_status(status="ЗАВЕРШЕНА")` из `deal.amount`.
- **Выплаты партнёрам** — что мы выплатили клиенту в USDT за его счёт. Авто-записывается из `deal.fee` если оно в USDT.
- **Стоимость ЛК** — что заплатили за купленные ЛК (в USDT).
- **Курс USDT** — закупочный (мы покупаем по нему) + партнёрский (по нему платим). Дельта = наш доход на обмене.
- **Ручные правки** — приход (+) или расход (−), вводятся менеджером.
- **Зарплаты** — авто из чистой маржи: операционист 15%, откупщик 2%.

## Авто-запись

При смене статуса сделки на `ЗАВЕРШЕНА`:
1. `accounting_add_turnover(deal_id, amount_rub, label=bank)`
2. Если `deal.fee` в USDT → `accounting_add_partner_payout(deal_id, amount_usdt, client)`

## Ручные команды (в чате «Бухгалтерия»)

| Команда | Что делает |
|---|---|
| `отчёт` | отчёт за сегодня |
| `отчёт 2026-05-04` | отчёт за конкретную дату |
| `курс 95 100` | закупочный 95 ₽ / партнёру 100 ₽ |
| `сумма 50000 #95941 описание` | оборот по сделке |
| `партнёр @client 400 #95941` | выплата партнёру в USDT |
| `лк Альфа 400` | купили ЛК Альфа за 400 USDT |
| `приход 5000 откат` | ручной приход +5000 ₽ |
| `расход 500 такси` | ручной расход −500 ₽ |
| `удали правку 2` | удалить ручную правку #2 |
| `помощь` / `?` / `/help` | подсказка |

```

💸 Партнёрам: 400.00 USDT (≈ 40 000 ₽)
🛒 ЛК: 200.00 USDT (≈ 19 000 ₽)
📝 Ручные правки: −500 ₽
💱 Курс USDT: закуп 95.00 ₽ / партнёру 100.00 ₽ (дельта: 2 000 ₽)

━━━━━━━━━━━━━━
💼 До зарплат: −9 500 ₽
🧾 Зарплата операциониста (15%): 0 ₽
🧾 Зарплата откупщика (2%): 0 ₽
✅ Чистая маржа: −9 500 ₽

  • #95941 — 50 000 ₽ Альфа
💸 Выплачено партнёрам:
  • #95941 @ivan_777 — 400 USDT
🛒 Куплено ЛК:
  • Альфа — 200 USDT
📝 Правки:
  [1] такси: −500 ₽
```
