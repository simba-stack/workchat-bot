## Процедура: Отработка аккаунтов (Перевяз ЛК)

Когда выполнен «Перевяз ЛК» — отправить в группу **Отработка аккаунтов** по шаблону:
```
Банк:
Номер сделки: (если есть; если нет — указать: выплата после отработки / на USDT TRC20)
Статус: В РАБОТЕ / УСПЕШНО ОТРАБОТАНО / ОТМЕНА СДЕЛКИ / БЛОК
Поставщик:
```

### Поведение юзербота — код в ЧРМ
Если клиент написал код в ЧРМ и он уже продублирован в чат — юзербот **не просит скинуть код повторно**, он уже там.

### Как юзербот понимает, что ЛК отработан
В конце каждого дня операционист скидывает список ЛК со статусами:
- **Отработан** — сделка отпускается
- **В работе** — сделка продолжается
- **Блок (сумма)** — требует внимания оператора
